# nmt_lab_shared

Gemeinsame Infrastruktur fuer die Projekte im `nmt_lab`-Workspace.

Der erste Baustein ist `nmt_lab_shared.run_config` mit den oeffentlichen
Funktionen `read_run_config(...)` und `write_run_config(...)` fuer
standardisierte `*_config.yaml`-Dateien mit Timestamp- und Git-Metadaten.
